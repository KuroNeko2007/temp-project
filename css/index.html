Components of a Computer System:
  1. Input Unit:
    • This unit takes input from the user with the help of any of the devices:
    • Keyboard
    • Mouse
    • Scanner
  2. Central Processing Unit:
    • This unit is responsible for controlling the units of a computer ... stores data temporarily while performing arithmetic and logical ... It has following sub-units:
    • Control Unit
    • ALU
    • Memory
  3. Output Unit:
    • This unit provides the user with the output which may be on:
    • Printer
    • VDU
    • Speaker
